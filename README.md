## Cara install

Download dulu

```bash
  download aja <code>, pilih yang Zip
```
<img width="460" height="399" alt="image" src="https://github.com/user-attachments/assets/cca31126-95df-45ce-a132-ad8588160da0" />


Extract file zip
```bash
  taruh beberapa file tersebut ke 1 folder yang sama
```

Install extension, gunakan chrome saja, aku gak jamin bisa di browser lain


Cara install

```bash
  paste ini  : chrome://extensions/ 
  lalu enter
  di pojok kanan ada developer mode, nyalakan
```
