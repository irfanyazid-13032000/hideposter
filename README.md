
# Hide Poster

sebuah tools untuk hide poster seat dan poster idol pose menggoda yang tidak penting itu sehingga kamu tidak perlu scroll kebawah karena akan memakan waktu
## Cara install

Download dulu

```bash
  download aja <code>, pilih yang Zip
```

Extract file zip
```bash
  taruh beberapa file tersebut ke 1 folder yang sama
```

Install extension, gunakan chrome saja, aku gak jamin bisa di browser lain


Cara install

```bash
  paste ini  : chrome://extensions/ 
  lalu enter
  di pojok kanan ada developer mode, nyalakan
  di pojok kiri atas ada load unpacked, klik
  pilih folder dari hasil extract-an tadi
```
